#Use git
###1.short operative

* 1.
```
git init
git add .
git commit -m'first commit'
```
the short operative is:

```
git commit -am'test comm'

```

* 2.
```
git add .
git commit -m'aa'
git push origin master

```

the short operative is:

```
git commit -am'test comm' && git push origin master
```
